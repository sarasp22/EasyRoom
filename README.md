# EasyRoom

EasyRoom est une application web qui facilite la recherche et la gestion de colocations. Elle permet aux utilisateurs de trouver des chambres disponibles, de publier des annonces et de gérer les échanges entre colocataires de manière simple et efficace.

---

## Objectif du projet

L’objectif d’EasyRoom est de simplifier la mise en relation entre personnes cherchant une chambre et celles proposant un logement, en offrant une plateforme claire, intuitive et centralisée.

---

## Fonctionnalités principales

* Consultation des annonces de chambres disponibles
* Publication d’une annonce de location
* Gestion des informations du logement
* Mise en relation entre utilisateurs
* Interface simple et intuitive

---

## Technologies utilisées

* **Frontend** : HTML, CSS, JavaScript
* **Backend** : Ruby on Rails
* **Base de données** : PostgreSQL
* **Outils** : Git, GitHub

---

## Installation et configuration

### Prérequis

* Ruby
* Rails
* PostgreSQL
* Git

### Étapes d’installation

Cloner le repository :

```bash
git clone https://github.com/username/easyroom.git
cd easyroom
```

Installer les dépendances :

```bash
bundle install
```

Créer et initialiser la base de données :

```bash
rails db:create
rails db:migrate
```

Lancer le serveur :

```bash
rails server
```

L’application est alors accessible à l’adresse : `http://localhost:3000`
